DICAS
as funções/o codigo é o mesmo em quase todos os SGBDs

-Função IsNull no SQLSERVEr = Função COALESCE no FireBird
	https://www.dio.me/articles/tratamento-de-campo-nulo-firebird
	A função COALESCE em SQL é usada para retornar o primeiro valor não nulo em uma lista de expressões. 
	Ela é especialmente útil quando você deseja obter o primeiro valor válido de uma série de colunas ou expressões,
	em vez de simplesmente retornar o primeiro valor encontrado,
	como aconteceria com a função IFNULL ou ISNULL (dependendo do banco de dados que você está usando).

	SELECT COALESCE(NomeAlternativo, Nome) AS NomeSelecionado, Descricao FROM Produtos;
	Nesse exemplo, se "NomeAlternativo" não for nulo para um determinado produto, esse valor será retornado como "NomeSelecionado".
	Caso contrário, o "Nome" principal será retornado.

	usar bastante com valores numericos
	retornar 0 caso seja nulo

-Função Top no SQLServer = Função First no FireBird
	https://razorsql.com/articles/firebird_select_top_syntax.html
	A sintaxe "select top" é uma sintaxe usada por certos bancos de dados para limitar as linhas retornadas às primeiras N linhas de uma consulta.
	O Firebird tem uma maneira de selecionar as N primeiras linhas de uma consulta SQL,
	mas a sintaxe não usa a palavra-chave top. Em vez disso, a palavra-chave "first" deve ser usada.

	SELECT FIRST 10 * FROM sua_tabela;
	Este comando irá retornar apenas as 10 primeiras linhas da tabela sua_tabela.
